# ChatGPT Electron wrapper

## Build and start

You need NodeJS installed.

```shell
npm install

# or

yarn start
```

Then start with one of these:

```shell
yarn start
npm start
bun start
```

## F.A.Q.

#### Why?

Because Alt+Tab is faster.

#### Does it keep authentication across sessions?

Yes.

#### Are links opened in the default desktop browser?

Yes.
